# LCart

A simple online shopping cart built with React , Material UI, Cart system built with useReducer and Context.

# Tech stack

- React (with hooks and context api)
- Material UI for styling and layout
- React Router for navigation
- Jest + React testing library for testing

# To Run Locally

## Data handling

The data for the products was stored locally in `src/data/inventory.json` and the app expects it
Please add the same inventory.json file that was provided with the assignment instruction
Once added the app will load and display the products

## Install dependencies

npm install

## Start developement Server

npm start

## Run Tests

npm test
